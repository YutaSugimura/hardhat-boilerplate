# hardhat

## Compile Contracts
```zsh
  npx hardhat compile
```

## Testing your contracts
```zsh
  npx hardhat test
```

## Deploying your contracts
```zsh
  npx hardhat run scripts/sample-script.js
```
## Starts a JSON-RPC server on top of Hardhat Network
```zsh
  npx hardhat node
```

## Sample Accounts
```zsh
  npx hardhat accounts
```
